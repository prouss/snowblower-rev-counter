Snowblower Tachometer

Welcome to the snowblower-rev-counter wiki!

In this project we want to design and build a tachometer (revolution counter) for a very old snowblower.

The goal of this project is not to build something useful, it is rather to let my 15 years old son acquire some knowledge in the following fields:

  • Basic analog and digital electronic design
  • Arduino programming
  • 3D design and printing
  • Planning a project and working on it from the beginning to the end
